Eclipse

Integrated Development Environment，IDE，集成开发环境
集成开发环境，是一种专门用来提高Java开发效率的软件。
免费的IDE当中：Eclipse
收费的IDE当中：IntelliJ IDEA

1.0 Eclipse的基本使用

1.1 选择工作空间

工作空间 其实就是我们写的源代码所在的目录

1.2 用Eclipse来完成一个HelloWorld案例

代码以项目为基本单位
创建项目
创建包
创建类
编写代码

1.3 编译

自动编译，在保存的那一刻帮你做好了

1.4 运行

点击虫子后面的绿色三角形按钮
点击Run菜单下的run。也可以使用快捷键Ctrl+F11
选择要运行的文件或者在要运行的文件内容中
右键 – Run as - Java Application即可
看到Console即可，它就是Eclipse自带的控制台

2.0 Eclipse视窗与视图

2.1 视窗

每一个基本的窗体被称为视窗

PackageExplorer 显示项目结构，包，类，及资源
Outline 显示类的结构，方便查找，识别，修改
Console 程序运行的结果在该窗口显示
Problems 显示所有语法及错误所在的位置
Hierarchy 显示Java继承层次结构，选中类后F4

2.2 视图

视图 是由某些视窗的组合而成的。

举例
Java视图
Debug视图

3.0 Eclipse中快捷键的使用

Ctrl+F11 程序运行

Alt+/ 代码提示
Ctrl+1 代码自动补全
Ctrl+Shift+F 代码格式化

Alt+Shift+R 批量修改
Ctrl+Shift+X/Y 大小写

Ctrl+/ 单行注释
Ctrl+Shift+/（增加）,Ctrl+Shift+\（取消） 多行注释

Alt+↓↑ 整行代码上下移动
Ctrl+Alt+↓ 复制当前行到下一行(复制增加)
Ctrl+Alt+↑ 复制当前行到上一行(复制增加)

Ctrl+Shift+O 导入包
Alt+shift+S+C 生成无参构造方法
Alt+shift+S+O 生成有参构造方法
Alt+shift+S+R 生成setxxx()和getXXX()方法

Alt+shift+A 一次生成多个成员变量
Alt+shift+S+S 重写toString
Alt+shift+S+H 重写equals
